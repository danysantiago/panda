Panda Code Reviews

Welcome to Panda Code Reviews. Panda Code Reviews is a free cloud application built by Team Aguacate for reviewing and grading studentsâ€™ code for computer science assignments written in Java.





Git Repositories


Panda integrates the use of repositories for assignments submissions. Panda automatically generates a git repository for each assignment for hosting the student's code.



Courses and Assignments

Professors can create courses. They can create assignments and design test cases that are automatically executed by Panda at the moment when a student submits his/her code.

Automatic Grading

When a student submits an assignment, Panda automatically produces a code quality report and grades the assignment based on the test cases.
