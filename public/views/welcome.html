Git Repositories

Panda integrates the use of repositories for assignments submissions. When there is a new assignment, Panda automatically generates a git repository for hosting the student's code. Submissions will be via git, so that professors do not need to worry about having a lot of assignments in their emails.

Courses and Assignments

Professors can create courses. They can create assignments and design test cases that are automatically executed by Panda at the moment when a student submits his/her code.

Automatic Grading

When a student submits an assignment, Panda automatically produces a code quality report and grades the assignment based on the test cases. The grade is not posted until the grader approves it, so the grade can be overwritten if desired.
